# 餐饮饮品行业适配

适用：餐厅、咖啡店、奶茶店、烘焙店、快餐店、小吃店、甜品店等。

## 1. 餐饮饮品常见经营问题

1. 午餐/晚餐/下午茶某个时段订单下降；
2. 周边竞品变多；
3. 出餐/出杯慢导致差评；
4. 菜单太多顾客不知道点什么；
5. 客单价低；
6. 食材成本上涨；
7. 低价活动订单多但不赚钱；
8. 外卖/堂食结构变化；
9. 高峰期忙乱，淡时段没人；
10. 新品卖不动或爆款不稳定。

## 2. 餐饮饮品优先指标

| 指标 | 用途 |
|---|---|
| 销售额 | 判断整体收入 |
| 订单数 | 判断顾客成交数量 |
| 客单价 | 判断套餐和加购效果 |
| 分时段订单 | 判断午餐、晚餐、下午茶、周末问题 |
| TOP商品 | 判断主推和爆款 |
| 套餐销量 | 判断客单和场景套餐 |
| 出餐/出杯时间 | 判断履约效率 |
| 差评关键词 | 判断等待、口味、卫生、服务问题 |
| 食材成本/毛利 | 判断是否赚钱 |

## 3. 无数据时替代指标

| 标准数据 | 替代方式 |
|---|---|
| 人流 | 路过人数、看菜单人数、进店人数 |
| 转化率 | 看菜单后进店、进店后下单 |
| 出餐效率 | 高峰期平均等待时间、排队长度 |
| 复购 | 熟客数量、员工识别、会员/外卖重复订单 |
| 口味反馈 | 评价关键词、顾客口头反馈 |

## 4. 回答时的行业化 To-do 示例

### 午餐下降

> 先看午餐时段订单是否下降高于全天平均。如果是，优先检查工作日午餐套餐、出餐速度和周边竞品午餐活动。

### 晚餐下降

> 先看双人/多人套餐、聚餐场景、晚餐评价和社媒展示。晚餐下降不一定是价格问题，可能是场景套餐不清楚或竞品更适合聚餐。

### 出餐/出杯慢

> 先统计高峰期平均等待时间。如果等待类差评近30天出现3次以上，先下调复杂商品高峰期主推，提前备料或调整排班。

### 客单价低

> 用主食/饮品/小食/甜品组合做套餐，不建议直接涨价。观察14天客单价和套餐销量。

## 5. 禁止泛化表达

不要只说：

- 提升口味；
- 优化出餐；
- 加强服务；
- 做活动引流。

要具体到：

- 哪个时段；
- 哪个套餐；
- 哪个商品；
- 哪个差评关键词；
- 哪个验证指标。
