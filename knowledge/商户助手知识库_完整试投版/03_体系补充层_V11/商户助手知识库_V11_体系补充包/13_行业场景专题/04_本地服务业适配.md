# 本地服务业适配

适用：洗衣、宠物服务、维修、健身、培训、家政、上门服务、预约类服务等。

## 1. 常见经营问题

1. 咨询多但成交少；
2. 预约爽约；
3. 服务周期长，复购慢；
4. 顾客不信任新服务；
5. 服务时间太长影响接待量；
6. 投诉和售后处理；
7. 老客提醒不及时；
8. 员工履约不稳定；
9. 价格解释困难；
10. 服务项目太复杂。

## 2. 优先指标

| 指标 | 用途 |
|---|---|
| 咨询量 | 判断需求 |
| 预约量 | 判断转化 |
| 到店/上门完成量 | 判断履约 |
| 爽约率 | 判断预约质量 |
| 服务时长 | 判断产能 |
| 客单价 | 判断套餐/升级 |
| 复购周期 | 判断提醒 |
| 投诉关键词 | 判断服务风险 |

## 3. 无数据替代指标

- 每天电话/私信咨询；
- 预约成功数量；
- 未成交原因；
- 爽约人数；
- 服务完成后顾客反馈；
- 老客多久回来一次。

## 4. 行业化 To-do 示例

### 咨询多但成交少

> 先记录顾客最常问的是价格、服务内容、时长、上门范围还是售后。如果多数人问完价格就走，说明价格价值感和服务说明不足。

### 复购周期长

> 不要频繁促销。按服务周期提醒，例如洗衣、宠物美容、维修保养、健身续课等，都应该在合适时间点触达。

### 预约爽约

> 先确认爽约是否集中在某类时段或服务项目。可增加预约确认、提前提醒、订金或候补机制。

## 5. 禁止泛化表达

不要只说：

- 提升服务；
- 做会员；
- 做预约管理。

必须说清：

- 哪个服务项目；
- 哪个预约时段；
- 哪类顾客；
- 提前多久提醒；
- 观察爽约率、成交率还是复购。
