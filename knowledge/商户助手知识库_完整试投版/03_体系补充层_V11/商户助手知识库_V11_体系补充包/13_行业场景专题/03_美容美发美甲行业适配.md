# 美容美发美甲行业适配

适用：美容院、美发店、美甲店、皮肤管理、按摩、护理类服务门店。

## 1. 常见经营问题

1. 咨询多但预约少；
2. 新客只做低价体验，不升级；
3. 服务项目太多，用户看不懂；
4. 顾客担心被强推；
5. 差评集中在服务、效果、推销、价格不清；
6. 老客周期性复购弱；
7. 员工服务不稳定；
8. 作品图或案例不足；
9. 预约爽约；
10. 次卡/会员设计不合理。

## 2. 优先指标

| 指标 | 用途 |
|---|---|
| 咨询量 | 判断兴趣 |
| 预约成功数 | 判断转化 |
| 到店率 | 判断爽约 |
| 新客体验数 | 判断拉新 |
| 升级项目销量 | 判断客单价 |
| 次卡/会员购买 | 判断长期复购 |
| 复购周期 | 判断提醒时机 |
| 服务时长 | 判断接待能力 |
| 差评关键词 | 判断服务风险 |

## 3. 无数据替代指标

- 私信/电话咨询数量；
- 咨询后预约人数；
- 到店但未消费原因；
- 顾客最常问价格、效果、时长还是安全；
- 服务后是否愿意评价；
- 老客通常多久回来一次。

## 4. 行业化 To-do 示例

### 咨询多但不预约

> 先检查价格是否清楚、作品图是否真实、服务时长和内容是否说明。如果顾客反复问价格，说明价格透明度不足，不要先做更多广告。

### 新客不升级

> 不要强推办卡。先设计“基础体验 + 14天内护理升级权益”，让顾客第二次回来再升级。

### 差评说强推

> 接待话术必须改成“先说明项目内容和价格，再给升级选择”。同时在预约页写清基础服务包含什么、不包含什么。

## 5. 禁止泛化表达

不要说：

- 提高服务质量；
- 强化会员运营；
- 推高端项目。

要说：

- 哪个项目作为低门槛体验；
- 哪个项目作为升级；
- 多少天内触达；
- 使用什么话术；
- 看预约转化、升级率还是差评变化。
