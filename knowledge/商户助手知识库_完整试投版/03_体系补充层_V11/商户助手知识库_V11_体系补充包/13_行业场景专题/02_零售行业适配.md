# 零售行业适配

适用：服装、美妆、便利、礼品、母婴、家居、小型综合零售等。

## 1. 零售常见经营问题

1. 进店多但成交少；
2. 试穿/试用多但购买少；
3. SKU太多，陈列混乱；
4. 库存积压；
5. 低价清仓影响利润；
6. 客单价低；
7. 复购不稳定；
8. 线上页面或社媒转化弱；
9. 新品卖不动；
10. 季节性库存压力。

## 2. 零售优先指标

| 指标 | 用途 |
|---|---|
| 进店人数 | 判断客流 |
| 试穿/试用/询价人数 | 判断兴趣 |
| 成交订单数 | 判断转化 |
| 客单价 | 判断组合和连带销售 |
| SKU销量 | 判断商品结构 |
| 库存周转 | 判断积压和补货 |
| 毛利 | 判断促销是否健康 |
| 退换货/投诉 | 判断商品质量和尺码/适配问题 |

## 3. 无数据时替代指标

- 员工记录每天进店人数、试穿/试用人数、购买人数；
- 记录顾客最常问的3个问题；
- 记录试了但不买的原因；
- 每周统计卖得最好和最差的SKU；
- 标记滞销、临期、过季商品。

## 4. 回答时的行业化 To-do 示例

### 试穿多但成交少

> 先判断是尺码、价格、搭配、库存还是员工推荐问题。让员工连续3天记录“试穿后不买”的原因，再决定是调陈列、改搭配、做第二件优惠还是补尺码。

### 库存积压

> 不要一开始全场清仓。先按库存金额、销量、毛利、季节性分层。高库存高毛利商品先做组合推荐，低库存低毛利可做清仓。

### 客单价低

> 用主商品 + 配件 / 第二件折扣 / 套装搭配提高客单。观察14天连带率和客单价。

## 5. 禁止泛化表达

不要只说：

- 优化陈列；
- 提高转化；
- 做促销清库存。

必须写清：

- 调整哪个货架；
- 哪些SKU降推荐；
- 哪些SKU做组合；
- 测试多久；
- 看连带率、客单价还是库存周转。
