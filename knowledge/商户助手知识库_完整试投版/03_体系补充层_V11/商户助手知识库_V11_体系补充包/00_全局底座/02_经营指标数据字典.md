# 经营指标数据字典

> 用途：统一所有问题卡中涉及的数据指标定义，避免内容同事和 AI 对同一指标理解不一致。

## 1. 核心销售指标

### 销售额 / 营业额

定义：门店在某个周期内产生的总收入。  
常见周期：日、周、月、活动前7天、活动期间、活动后7天。  
用途：判断整体收入表现。

注意：销售额下降只是结果，不是原因。必须继续拆订单数和客单价。

---

### 订单数

定义：门店在某个周期内成交的订单数量。  
用途：判断成交人数或成交频次是否变化。

常见判断：

| 表现 | 可能含义 |
|---|---|
| 销售额下降，订单数下降 | 顾客数量或成交减少 |
| 销售额稳定，订单数上升 | 可能客单价下降 |
| 订单数上升但利润下降 | 低价订单或活动折扣过重 |

---

### 客单价 / ATV

计算：

```text
客单价 = 销售额 ÷ 订单数
```

用途：判断每单消费金额是否变化。

常见判断：

| 表现 | 可能含义 |
|---|---|
| 客单价下降5%-10% | 低价商品占比上升，需关注 |
| 客单价下降10%以上 | 套餐、高价商品、加购可能出问题 |
| 客单价上升但订单下降 | 可能新客减少，剩余顾客买得多 |
| 订单上升但客单下降 | 活动或低价商品占比过高 |

---

## 2. 利润与成本指标

### 毛利

基础定义：销售额减去商品或服务的直接成本。  
用途：判断订单是否真正赚钱。

如果商户没有精确毛利，可先用“高 / 中 / 低”粗略分类。

| 毛利状态 | 可用表达 |
|---|---|
| 高毛利 | 适合做利润款、加购、套餐升级 |
| 中毛利 | 适合主推和稳定成交 |
| 低毛利 | 适合引流，但不能无限放大 |

---

### 毛利率

计算：

```text
毛利率 = 毛利 ÷ 销售额
```

用途：判断利润结构是否健康。

常见判断：

- 订单增加但毛利率下降：活动或低价商品占比过高；
- 高销量商品毛利率低：需要加购或套餐承接；
- 毛利率下降且差评增加：可能低价活动带来履约压力。

---

## 3. 商品与菜单指标

### TOP 商品

定义：某个周期内销量最高或销售额最高的商品。  
用途：判断核心商品和主推方向。

应区分：

- 销量TOP；
- 销售额TOP；
- 毛利TOP；
- 差评TOP。

---

### 套餐销量

定义：套餐类商品的销售数量。  
用途：判断套餐是否有效承接客单价和多人消费。

常见判断：

| 表现 | 可能含义 |
|---|---|
| 套餐销量下降，单品销量上升 | 商品结构变差，客单可能受影响 |
| 套餐销量上升，客单价上升 | 套餐设计有效 |
| 套餐销量上升，毛利下降 | 套餐折扣过重或组合不合理 |

---

### 加购率

定义：订单中有加购商品或服务升级的比例。  
如果没有系统数据，可用“员工观察 + 加购品销量”替代。

用途：判断是否能提高客单价。

---

## 4. 顾客指标

### 新客

定义：第一次消费或第一次到店/下单的顾客。  
无系统时可用：新面孔、首次预约、首次留资、首次下单估算。

---

### 老客

定义：重复消费的顾客。  
无系统时可用：熟客数量、重复预约、会员使用、员工识别。

---

### 复购

定义：顾客再次消费。  
不同业态复购周期不同：

| 行业 | 复购周期参考 |
|---|---|
| 奶茶/咖啡 | 几天到几周 |
| 餐饮 | 几天到几周，聚餐类更长 |
| 美甲/美容 | 2-6周或更长 |
| 洗衣/宠物服务 | 周期性服务，通常数周到数月 |
| 零售 | 取决于品类，可能数周到数月 |

注意：不能把“短期没回来”直接判断为流失，必须结合行业周期。

---

## 5. 评价与口碑指标

### 评分

定义：公开平台或内部评价系统中的门店综合评分。  
用途：判断新客信任度。

注意：评分不能单独解释经营结果，需要结合评价数量、差评关键词、竞品评分。

---

### 差评关键词

常见分类：

| 关键词 | 可能问题 |
|---|---|
| 等太久、排队久 | 高峰期履约效率问题 |
| 服务不好、态度差 | 员工培训或服务流程问题 |
| 贵、不值、份量少 | 价格价值感不匹配 |
| 和图片不一样 | 展示与实际不一致 |
| 不新鲜、不干净 | 产品或卫生风险 |
| 不知道点什么 | 菜单结构不清楚 |

判断标准：

- 同类问题出现1次：记录观察；
- 2-3次：检查流程；
- 3次以上：作为经营问题处理；
- 多平台重复出现：优先级最高。

---

## 6. 线下替代指标

很多线下商户没有人流、点击率、转化率，应使用替代指标。

| 标准指标 | 线下替代指标 |
|---|---|
| 曝光 | 路过人数、看店人数、平台浏览、社媒浏览 |
| 点击率 | 看菜单人数、询价人数、私信、电话咨询 |
| 转化率 | 进店成交、预约成功、咨询后购买 |
| 客单价 | 每单金额、套餐占比、加购数量 |
| 复购率 | 熟客数量、重复预约、老客占比 |
| 口碑 | 评分、差评关键词、好评关键词、转介绍 |
| 履约能力 | 等待时间、缺货、排队、员工反馈 |

---

## 7. 活动复盘指标

活动不能只看订单。

必须看：

1. 活动前7天订单和销售额；
2. 活动期间订单和销售额；
3. 活动后7天订单和销售额；
4. 活动商品毛利；
5. 新客数量；
6. 客单价变化；
7. 差评变化；
8. 活动后复购。

常见判断：

| 结果 | 判断 |
|---|---|
| 订单涨，毛利稳 | 活动相对健康 |
| 订单涨，毛利大降 | 折扣过重 |
| 新客多，复购少 | 缺少承接 |
| 订单涨，差评多 | 接待能力不足 |
| 活动后快速回落 | 没有留存动作 |
