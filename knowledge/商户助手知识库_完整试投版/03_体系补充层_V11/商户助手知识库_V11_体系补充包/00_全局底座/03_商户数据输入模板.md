# 商户数据输入模板

> 用途：让商户更容易提供信息，降低“上传完整数据”的心理门槛。  
> 原则：先要低敏、汇总、大概范围，再逐步引导更精确数据。

## 1. 通用经营诊断模板

适用于：生意下降、销售额下降、不知道问题出在哪里。

```text
最近7天销售额：
上一个7天销售额：
最近7天订单数：
上一个7天订单数：
最近7天客单价：
上一个7天客单价：
最近卖得最好的5个商品：
最近卖得变差的商品：
近30天差评关键词：
周边最近是否有新竞品/活动：
```

如果商户没有精确数据，可让其填写：

```text
销售额：上升 / 持平 / 下降，约下降多少？
订单数：上升 / 持平 / 下降，约下降多少？
客单价：上升 / 持平 / 下降，约下降多少？
最近顾客常问或抱怨什么？
```
