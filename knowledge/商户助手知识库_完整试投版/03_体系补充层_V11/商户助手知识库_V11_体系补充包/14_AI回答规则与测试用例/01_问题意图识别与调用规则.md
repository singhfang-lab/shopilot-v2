# 问题意图识别与调用规则

> 用途：解决“商户问一个问题，AI 到底该调用哪张问题卡”的问题。

## 1. 调用原则

AI 不能只按关键词匹配。必须综合判断：

1. 商户问题的核心意图；
2. 问题类型；
3. 数据状态；
4. 商户行业；
5. 是否涉及多个模块；
6. 是否需要主卡 + 辅助卡组合。

---

## 2. 问题类型识别

| 商户问法 | 问题类型 | 主回答方式 |
|---|---|---|
| 为什么下降？为什么不好？ | 原因诊断型 | 列影响因素、定位问题、给排查/诊断 |
| 要不要做？该不该降价？ | 判断决策型 | 给判断条件、风险、小范围测试 |
| 怎么做？怎么提高？ | 方案生成型 | 给方案结构、执行步骤、验证标准 |
| 怎么看？怎么分析？ | 方法教学型 | 教记录、观察、判断方法 |
| 有没有效果？该不该继续？ | 复盘评估型 | 看数据、判断有效/无效、给下次方案 |
| 我怎么不一样？怎么表达？ | 定位表达型 | 目标顾客、差异化、卖点、表达落地 |

---

## 3. 主卡调用规则

### 生意下滑类

| 商户问题 | 主卡 |
|---|---|
| 最近生意变差了 | 最近生意变差了，我应该先排查什么？ |
| 销售额下降 | 销售额下降，是订单问题还是客单价问题？ |
| 订单上升但利润下降 | 订单上升但利润下降，问题在哪里？ |
| 多家门店只有一家下降 | 多家门店中只有一家下降，为什么？ |

### 竞品类

| 商户问题 | 主卡 |
|---|---|
| 周边竞品很多 | 周边竞品很多，我应该怎么突围？ |
| 竞品便宜 | 竞品比我便宜，我要不要跟着降价？ |
| 竞品活动多 | 竞品活动很多，我该怎么应对？ |
| 想找差异化 | 如何找差异化定位？ |

### 产品菜单类

| 商户问题 | 主卡 |
|---|---|
| 菜单太多 | 菜单太多，顾客不知道点什么，怎么办？ |
| 设计套餐 | 客单价太低，怎么通过套餐提高？ |
| 选爆款 | 如何判断一个商品适不适合做爆款？ |
| 商品卖得差 | 商品卖得差，要不要下架？ |

### 顾客类

| 商户问题 | 主卡 |
|---|---|
| 新客不回来 | 新客来了但不复购，怎么办？ |
| 老客少 | 如何提高老顾客回头率？ |
| 差评多 | 差评变多，会不会影响新客？应该怎么处理？ |
| 评分低 | 评分不高，怎么提升口碑评分？ |

### 活动类

| 商户问题 | 主卡 |
|---|---|
| 低成本引流 | 小商户预算有限，怎么做低成本引流？ |
| 节日活动 | 节假日活动怎么做才不亏钱？ |
| 活动效果 | 活动做完后，怎么判断有没有效果？ |
| 活动不赚钱 | 活动订单涨了但不赚钱，怎么办？ |

## 4. 特殊问题调用规则

### 场景：同价位竞品增加导致下降

商户问：

> 我是韩料店，附近开了几家同价位竞品，营业额下降，怎么办？

主卡：

- 同价位竞品变多，营业额下降怎么办？（若没有此卡，可用“周边竞品很多，我应该怎么突围？”）

辅助卡：

- 销售额下降，是订单问题还是客单价问题？
- 如何找差异化定位？
- 如何分析周边竞争对手？
- 如何设计新客活动？

### 场景：营收稳定但想增长

主卡：

- 营收稳定但想继续增长，应该从哪里突破？

辅助卡：

- 客单价太低，怎么通过套餐提高？
- 如何设计引流款、招牌款和利润款？
- 如何做周边合作引流？
- 社媒不知道发什么，怎么做基础内容？

---

## 5. 调用数量控制

1. 每次回答只允许 1 张主卡；
2. 辅助卡最多 2-3 张；
3. 不能把所有相关知识都堆进回答；
4. 如果问题很复杂，先聚焦商户最急的问题。

---

## 6. 调用失败的表现

如果回答出现以下情况，说明问题卡调用可能错误：

- 商户问复盘，AI却给活动方案；
- 商户问竞品，AI直接讲社媒；
- 商户问单店下降，AI直接讲品牌整体；
- 商户问增长，AI没有先判断增长杠杆；
- 商户没有数据，AI直接下强结论。
