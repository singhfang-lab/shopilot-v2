# 商户定位画像卡

用于记录商户的长期定位信息，供问题卡回答时调用。定位画像不是替代经营数据，而是帮助 AI 判断差异化、目标顾客、社媒内容、活动风格和表达口径。

- 城市/商圈：
- 主要销售渠道：线下到店 / 外卖 / 预约 / 社媒 / 其他

## 2. 目标顾客
- 主要服务谁：
- 典型消费场景：工作日午餐 / 下午茶 / 约会 / 家庭 / 美容护理 / 快速服务 / 其他
- 预算区间：
- 顾客最在意的问题：价格 / 速度 / 体验 / 品质 / 信任 / 便利 / 其他

- 顾客为什么选你：
- 这个卖点是否有证据：好评关键词 / 复购 / 评分 / 招牌商品 / 案例
