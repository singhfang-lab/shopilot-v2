## 发现的问题

### 🔴 高优先级
1. **长回答被截断** — 超过 ~2000 字时末尾内容丢失（表格/排期不完整）
2. **多轮对话后期数据引用减弱** — 第3轮 data_usage 评分明显低于第1轮

### 🟡 中优先级
3. **SQL Agent 响应延迟高** — 平均 45s/轮，最高 57s
4. **E2E 注册跳转偶发不稳定** — cookie 写入与页面跳转时序竞争
5. **Admin Prompt 接口返回 200 而非 201** — 不符合 RESTful 规范

### 🟢 低优先级
6. **未绑定商家的去重逻辑缺失**
7. **无数据场景 AI 默认 IDR 单位** — 地区适配性问题
